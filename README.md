# Analisador de Sentimentos com Python

Aplicação que analisa frases digitadas pelo usuário e classifica
o sentimento como positivo, negativo ou neutro.

O projeto utiliza:

- Python
- TextBlob para análise de sentimento
- Tkinter para interface gráfica
- Matplotlib para geração de gráficos

## Tecnologias utilizadas

- Python
- TextBlob
- Tkinter
- Matplotlib

## Funcionalidades

- Interface gráfica
- Análise de sentimento
- Contador de sentimentos
- Visualização de gráfico

## Instalação

1 - Clone o repositório

git clone https://github.com/Pedro-henrique-1997/analise-sentimento-python

2 - Entre na pasta do projeto

cd analise-sentimento-python

3 - Instale as dependências

pip install -r requirements.txt

4 - Execute o programa

python main.py
